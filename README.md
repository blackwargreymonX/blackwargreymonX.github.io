# Duo Xu Homepage

This is Duo Xu's personal academic homepage, built with the AcadHomepage/Jekyll template and adapted from the resume content.

## Local Preview

```bash
bundle exec jekyll serve
```

The current local preview is running at:

```text
http://127.0.0.1:4010
```

## Main Files

- `_config.yml`: site metadata and sidebar profile.
- `_pages/about.md`: homepage content.
- `_data/navigation.yml`: top navigation anchors.
- `files/resume_xuduo.pdf`: downloadable resume.
